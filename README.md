PawFinder
 PawFinder is an online platform for pet adoption that links individuals seeking pets with animal shelters.With separate user and admin responsibilities, it offers an easy-to-use interface for maintaining shelters and ratings, exploring pets, and submitting adoption requests. To begin utilising the application, just open the index.html file in your web browser.

User
    View the pets that are available.
    See thorough pet profiles
    Send in and oversee adoption requests
    Create and read reviews
     Modify your own profile

The administrator:
    Go to the administrator's dashboard.
    See and handle every adoption request.
    Accept or deny petitions for adoption
    Oversee users, shelters, and pets.
    Keep an eye on platform statistics

 Features for Users:
    Browse Pets: Look for and select animals by breed, age, or shelter.
    View comprehensive pet profiles, including breed, size, and age.
    Adoption Requests: Send in and monitor applications for adoption
    Shelters Directory: See all affiliated shelters along with their contact details
    Reviews: Read adoption stories and provide comments.
    Manage your profile and update your personal information.


 Features for the Administrator
    Dashboard: A summary of all pets, users, and unfulfilled requests
    Request Management: Accept or deny adoption requests
    Pet management: Adding, changing, or getting rid of animals
    User Management: Access and control user profiles
    Shelter Management: Revise or delete information about partner shelters
